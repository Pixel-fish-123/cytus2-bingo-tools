# cytus2-bingo-
支持自定义曲库，自定义任务，ban曲和bingo生成功能

萌新杯裁判辅助系统ver0.1

song_package.xlsx 为本次比赛所需要的所有曲包，可按照格式自定义调整

打开程序后，显示“Successfully loaded all songs”时说明导入曲库成功

若失败，将错误代码发给Pixel_fish

按下1并回车，文件夹下会出现SelectedSongs.xlsx这个文件夹，该文件夹下共有16首歌曲，为本轮的抽取歌曲;

生成SelectedSongs.xlsx后，请按3关闭程序，打开文件，确认本轮抽取歌曲

裁判需要手动删除两首歌曲，作为本轮ban曲目，注意歌曲之间一定不能有空格！！！

打开文件的时候请关闭程序，保证文件已经保存完毕且WPS关闭后再次启动程序

按下2并回车，文件夹下会生成一个bingo.txt，把里面的内容复制到bingosync的自定义json处即可！

常见错误：

未关闭wps使用该程序

导出bingo时歌曲数量不为14,会出现错误报告“歌曲数量不正确”

若有其他bug请发给Pixel_fish
